\begin{slide}
\slidetitle{What is a MPL?}
\small
Mathematical (algebraic) modeling languages allow to formulate a
linear/nonlinear/integer mathematical programming problem in 
a way similar to the original mathematical formulation and 
automatically generate data for a corresponding solver.

\normalsize Uses for MPL's

\small
\begin{itemize}
\item Teaching modeling
\item Rapid prototyping
\item Projects with industry
\end{itemize}

\end{slide}

\begin{slide}
\slidetitle{What is different in Zimpl?}

Not integrated with the solver (comming)
No integrated windows environment
(possible if somebody has time for it)
Not the boss, but just a reader/translator
Not procedural
Only a subset of the possibilities of, e.g. AMPL.

\end{slide}
